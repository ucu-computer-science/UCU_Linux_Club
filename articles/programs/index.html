How to install YayðŸ”—

In the terminal emulator

sudo pacman -S git
cd ~/Downloads
git clone https://aur.archlinux.org/yay-bin.git
cd yay-bin
makepkg -si

Also, you need to add this to the end of pacman.conf file

sudo nvim /etc/pacman.conf

[archlinuxfr] </br>
SigLevel = Never </br>
Server = http://repo.archlinux.fr/$arch </br>
Install programs using yayðŸ”—
yay 'program_name'

Install zsh instead of bash and make it defaultðŸ”—

yay zsh
chsh -s $(which zsh)
